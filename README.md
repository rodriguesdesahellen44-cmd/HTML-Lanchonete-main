# HTML-lanchonete-main

Projeto frontend "Lanchonete Sabor Caseiro" — página estática em HTML/CSS/JS para adicionar itens ao pedido, salvar pedidos localmente (localStorage), listar clientes e ver últimos pedidos.

Como usar

1. Abra `index.html` em um navegador (ou rode um servidor simples: `python -m http.server 8000`).
2. Adicione itens do cardápio ao pedido.
3. Clique em "Abrir janela do App", preencha o nome do cliente e salve o pedido — os dados ficam no localStorage do navegador.

Arquivos

- `index.html` — marcação e modal para salvar pedidos
- `style.css` — estilos
- `index.js` — lógica de adicionar/remover itens e salvar pedidos

Observações

Este repositório contém apenas o front-end. O botão para executar o backend Python foi substituído por uma interface que salva pedidos localmente.
